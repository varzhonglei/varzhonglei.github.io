数组操作与日期对象

数组老是忘的操作

  • arr.forEach可以循环遍历数组每一位元素
    但是不返回值

不可以通过直接修改item来修改原数组，因为item只是原始数据的浅拷贝

  • arr.map可以循环遍历数组每一位元素，并且返回一个新的数组

  • arr.indexOf获取元素在该数组中的第一个匹配序列号，未匹配则返回-1

  • arr.findIndex获取元素在该数组中的第一个满足参数函数序列号，未匹配则返回-1

  • arr.find获取元素在该数组中的第一个满足参数函数的值， 未匹配则返回 undefined

  • arr.filter按条件过滤函数，返回true保留该item

  • arr.some arr.every 遍历数组 看数组中每一项值是否满足要求 返回布尔值

日期对象的一些操作

创建一个日期对象

var now = new Date()
var oneDay = new Date('2018-2-2')
var oneDay = new Date(2017,2,2)
var oneDay = new Date( 日期对象 ) 拷贝，两个对象互不影响

日期对象常用方法

oneDay.getFullYear 年
oneDay.getMonth 月，0~11
oneDay.getDate 日
oneDay.getDay 星期，返回0~6
oneDay.getTime 返回到1970-1-1日的毫秒速

遇到过的需求

  • 求出两个日期对象所相差的天数

  • 让当前日期对象加一天
