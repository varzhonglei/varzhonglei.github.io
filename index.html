set

  • 简单说明： set是类似与数组的数据结构，该集合中的元素不可重复

set 实例的属性和方法

  • Set 结构的实例有以下属性。
  1. Set.prototype.constructor：构造函数，默认就是Set函数。
  2. Set.prototype.size：返回Set实例的成员总数。 ---- 类似与数组.length

  • Set 实例的方法分为两大类：操作方法（用于操作数据）和遍历方法（用于遍历成员）。

  • 下面先介绍四个操作方法。

  1. add(value)：添加某个值，返回 Set 结构本身。 --- 可以链式调用
  2. delete(value)：删除某个值，返回一个布尔值，表示删除是否成功。
  3. has(value)：返回一个布尔值，表示该值是否为Set的成员。
  4. clear()：清除所有成员，没有返回值。
  • Set 结构的实例有四个遍历方法，可以用于遍历成员。
  1. keys()：返回键名的遍历器
  2. values()：返回键值的遍历器 -----默认遍历器
  3. entries()：返回键值对的遍历器
  4. forEach()：使用回调函数遍历每个成员
  • 特别指出: Set的遍历顺序就是插入顺序; 该点应用：用Set保存回调函数列表， 可按保存的顺序执行
  • Set结构默认可遍历
    Set.prototype[Symbol.iterator] === Set.prototype.values//true

set 应用

  1. 用Set保存回调函数列表， 可按保存的顺序执行
  2. 数组去重

  • 这里复习下，es5的经典数组去重

  • 点评： 这种方法通过对象的键名不可重复；从而达到去筛选重复值的效果
    依赖[]运算符， 在es5中(es6的Symbol也可以做键名) []运算符会把一切类型的值转化为字符串
    这意味着 当数组里有多个NaN时， NaN不会被去重，数组中所有的对象只被保留一个

  • 用与一般元素值去重足够了

  • 接下来看看es6 如何优雅去重

  • 点评： es6的去数组去重，强大了。

  • 当面临有多个NaN要加入Set集合时，只有一个 NaN 能被纳入集合。
    Set的不重复规则：使用的算法叫做“Same-value-zero equality”类似与 === ，与===区别： 该算法认为NaN等于NaN
  • 数组中的对象也可以正常去重了 注意 [{},{}] 中这两个对象是不同的，不会被去重
  1. 用Set实现集合的 并集（Union）、交集（Intersect）和差集（Difference）

  • 其他操作技巧： 如果想在遍历操作中，同步改变原来的 Set 结构，目前没有直接的方法，但有两种变通方法。

数组操作与日期对象

数组老是忘的操作

  • arr.forEach可以循环遍历数组每一位元素
    但是不返回值

不可以通过直接修改item来修改原数组，因为item只是原始数据的浅拷贝

  • arr.map可以循环遍历数组每一位元素，并且返回一个新的数组

  • arr.indexOf获取元素在该数组中的第一个匹配序列号，未匹配则返回-1

  • arr.findIndex获取元素在该数组中的第一个满足参数函数序列号，未匹配则返回-1

  • arr.find获取元素在该数组中的第一个满足参数函数的值， 未匹配则返回 undefined

  • arr.filter按条件过滤函数，返回true保留该item

  • arr.some arr.every 遍历数组 看数组中每一项值是否满足要求 返回布尔值

日期对象的一些操作

创建一个日期对象

var now = new Date()
var oneDay = new Date('2018-2-2')
var oneDay = new Date(2017,2,2)
var oneDay = new Date( 日期对象 ) 拷贝，两个对象互不影响

日期对象常用方法

oneDay.getFullYear 年
oneDay.getMonth 月，0~11
oneDay.getDate 日
oneDay.getDay 星期，返回0~6
oneDay.getTime 返回到1970-1-1日的毫秒速

  • 求出两个日期对象所相差的天数
